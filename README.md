# AI 長期投資配置顧問

AI 輔助長期投資配置的互動式網站原型。以公開市場資料為基底,協助使用者輸入持股、判斷配置是否需要調整。

## 功能
- 儀表板:總資產、YTD、配置偏離、AI 信心分數
- 持股管理:手動 / 券商匯入 / CSV
- AI 配置建議:推理過程 + 執行清單
- 調整時機提醒:觸發條件、信心分析、影響預估
- 資料基底來源:總經 / 央行 / 財報 / 新聞 / 技術 / 情緒
- 歷史回測、對話 AI、個人設定
- Tweaks 面板:風險等級 / 深淺主題 / 資料密度

## 線上展示
部署後網址:`https://rogerjjr.github.io/ai-investment-advisor/`

## 本地開發
因為是純靜態 HTML + JSX(Babel 瀏覽器端轉譯),用任何靜態伺服器即可:
```bash
python3 -m http.server 8000
# 開啟 http://localhost:8000
```

## 免責聲明
本網站所有建議皆為 AI 模型推論與 mock 資料,**不構成投資建議**。投資有風險,過去績效不代表未來結果。
